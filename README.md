# designskills-hub

**The [designskills.xyz](https://designskills.xyz) registry, as an installable skill.** Install it once and your AI agent knows how to search, evaluate, install, and buy portable design skills — the whole registry becomes part of your agent's vocabulary.

```
npx skills add VaporAviator/designskills-hub-skill
```

What your agent learns:

- **Search** the registry in plain use-case language — `pitch deck`, `warm minimalist`, `小红书` all work (`/api/search`)
- **Evaluate** before installing: `dna` style fingerprints, 7-dimension `securityScan`, ratings, stars
- **Install** via each skill's authoritative `installCmd` (plus a no-npx fallback), and report installs to the leaderboard
- **Buy** paid skills over [x402](https://x402.org) (USDC on Base) — always with per-purchase user approval, never silently
